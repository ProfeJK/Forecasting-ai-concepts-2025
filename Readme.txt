# 📊 Conceptos y Recetas para Forecasting y Analítica de Datos (2025)

Este repositorio contiene un **cuaderno en Python/Colab** con explicaciones, ejemplos y recetas prácticas para **predicción y forecasting en series temporales**, combinando estadística, machine learning y deep learning.

## 🚀 Contenido
- **Conceptos fundamentales**: correlación, autocorrelación, correlación cruzada, correlación parcial (ACF y PACF).
- **Pruebas de estacionariedad**: ADF, KPSS, descomposición de series.
- **Selección y extracción de características**: RFE, búsqueda exhaustiva, Lasso, Ridge, PCA, Autoencoders.
- **Recetas ganadoras para forecasting**: desde PCA+Lasso hasta ensembling, modelos híbridos y transformers.
- **Diagramas de decisión**: guía visual para elegir la estrategia adecuada según tus datos.
